== Description ==

A quicklook into 'Inline Crypto Price' Plugin:

	★ Easily insert live cryptocurrency rates for top currencies (BTC, ETH, XRP, LTC Binance Coin and more)
	★ Include Exchange rates for top fiat currencies (USD, GBP, EUR, AUD and more) 
	★ Easy to use short code inline with text in post articles or pages.
	★ Display crypto price cards - allow you to present real-time data to your audience in form of blocks widget!
	
This plugin uses 3rd party api service from coinmarketcap.com to fetch crypto rates.	

= Features in Detail =

'Inline Crypto Price' plugin allows you to display live exchange rates of top crypto currencies in multiple fiat currencies. You can insert price label inline with text of your post content or articles. Everything with simpleshort codes.

This plugin use coinmarket cap exchange rates and other crypto data.

== Frequently Asked Questions ==

= How to insert crypto exchange rate in my post article =

You can insert crypto prices in your post articles as plain text using shortcode [ICP-BTCUSD]. This shortcode is different based on your fiat currency. For example, to display rate for BTC to GBP, shortcode will be [ICP-BTCGBP].

you can get all currency codes from plugin menu.

= How to use price cards ? (PRO) =

You can insert price card for any crypto currency using shortcode [PRICE-CARD-ICP-BTCUSD].

= I have 2 shortcodes placed adjacently, will they work? =

Yes, you can use multiple times same shortcodes on same page. Please use a whitespace to separate 2 shortcodes.
